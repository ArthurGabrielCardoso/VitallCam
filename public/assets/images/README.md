# 📁 Assets da Clínica

## 🖼️ Imagens

### Logo da Clínica
Coloque o logo da sua clínica nesta pasta com o nome:
- `logo.png` (formato PNG recomendado)
- `logo.svg` (formato SVG também suportado)

### Especificações do Logo:
- **Formato**: PNG ou SVG
- **Tamanho recomendado**: 200x60px
- **Fundo**: Transparente (PNG) ou adaptável
- **Qualidade**: Alta resolução para displays retina

### Como usar:
1. Coloque seu arquivo de logo nesta pasta
2. Renomeie para `logo.png` ou `logo.svg`
3. O sistema detectará automaticamente o arquivo
4. O logo aparecerá na sidebar e cabeçalhos

### Outras imagens:
Você também pode colocar outras imagens da clínica aqui:
- `favicon.ico` - Ícone do navegador
- `background.jpg` - Imagem de fundo (se necessário)
- `placeholder.png` - Imagem padrão para pacientes

---

**💡 Dica**: Mantenha os arquivos de imagem otimizados para web para melhor performance!
